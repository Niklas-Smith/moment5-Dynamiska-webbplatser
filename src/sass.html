Hur det gått att använda SASS/SCSS

Tycker det varit intressant att arbeta med SASS/SCSS.
Jag gillar att man kan dela upp css-filen i flera partials
och detta kommer nog bara bli viktigare när arbetet blir större.
Även variabler känns logiska att använda för underhålla hemsidan i efterhand.
mixins/inheritance kommer nog kännas bättre ju mer man använder dom.
Slutsats: tycker inte SASS/SCSS gör något sämre än CSS så känns som ett logiskt steg framåt.

För- och nackdelar med SASS/SCSS

Några av de fördelar jag ser med sass

  • Kan använda nesting vilket gör att din kod ser bättre ut
  • Variabler gör att det är lättare att ändra i efterhand
  • Kunna använda partials vilket underlättar för att hitta och använda kod
  • Extra funktioner (minxins) att använda som inte finns i CSS.

Några av de nackdelar jag se med sass

  • Den enda jag ser är att man måste lära sig ett nytt hjälpmedel
